# Getting Start

## Requirement
- git
- foundry

## QuickStart
```
git clone https://github.com/kkontheway/Mini-Exploits-Playground.git
cd Auditor-Playground
forge install
```

## Test
```solidity
forge test
```

## Running a single test
```
forge test --mt test_functionName
```
